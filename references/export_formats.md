# Export Formats

## EndNote

Prefer RIS or ENW. Include title, journal, year, DOI, PMID, URL, abstract summary, and keywords when available.

## Zotero

Prefer BibTeX, RIS, or CSL-JSON for file import. Use Zotero Web API only when the user supplies OAuth/API credentials.

## NoteExpress

Prefer RIS or BibTeX. Keep DOI and journal fields clean because NoteExpress import quality depends heavily on standard tags.

## Internal Paper Shape

Use these fields when adding exporters:

- `pmid`
- `doi`
- `title_en`
- `journal`
- `year`
- `authors`
- `abstract_summary_en`
- `source_urls`
- `keywords`
